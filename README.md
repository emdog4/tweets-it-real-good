# tweets-it-real-good

sign in with twitter implementation
with mongo backed session store

prereqs.:
node.js and npm
mongod

create .env file in project root with the following
APP_COOKIE_SECRET=
TWITTER_CONSUMER_KEY=
TWITTER_CONSUMER_SECRET=

start mongod

cd into cloned dir
npm install
node ./bin/www

visit localhost:3000 in your browser
